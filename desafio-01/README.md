# Descrição

Criar um programa na linguagem Ruby que dê a saída do nome completo do usuário e a sua idade

## Desafio

Você precisa criar um programa simples de acesso a leitores em uma biblioteca e para isso o usuário deverá digitar seu nome e o sobrenome, além de sua idade que é um dado relevante para o departamento de marketing daquele local. Tudo isso, impresso, em uma única frase.

## Setup

- Instale as dependecias com ```bundle install```
- executa os test ```rspec```
